A Wordpress plugin that adds a new element to Breakdance builder for fancy button effects

== Description ==

24 buttons effects in one element for Breakdance

Upgrade to <a href="https://phox-elements.com">Phox Elements</a> to unlock 29 new elements for Breakdance.


== Installation ==

1. Upload the plugin folder to your /wp-content/plugins/ folder.
2. Go to the **Plugins** page and activate the plugin.
3. Open the breakdance page builder and select the Fancy Button element
